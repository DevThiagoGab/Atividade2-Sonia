# Atividade2-Sonia

Etapa 1 da Atividade (Descrever): 
1. Por que o arquivo .env não pode ser versionado? O arquivo .env não pode ser versionado porque ele guarda senhas e chaves secretas; enviá-lo para o GitHub expõe os dados e coloca a segurança do projeto em risco.
2. Qual a função do pip freeze no contexto de desenvolvimento em equipe? A função do pip freeze é listar todas as bibliotecas (e suas versões exatas) que você instalou no seu projeto.
3. O que pode acontecer se a SECRET_KEY for exposta? Se a SECRET_KEY for exposta, a segurança de todo o sistema é comprometida pois ela é a "assinatura digital" que valida tudo o que acontece no seu site.
4. Qual o papel do gitignore em projetos colaborativos? Dizer ao Git quais arquivos ou pastas ele deve ignorar e nunca enviar para o GitHub.
